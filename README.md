&gt; **Real-time analytics for last-mile delivery optimization**
&gt; 
&gt; Built to demonstrate alignment with [Bettermile's](https://bettermile.com) mission 
&gt; to transform parcel logistics through AI-powered route optimization and ETA prediction.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://delivery-insights-dashboard-rhza7pprwolkrqzx35gkfv.streamlit.app/)

## 🎯 What This Project Does

This interactive dashboard simulates a last-mile delivery operation with:
- **10,000+ simulated deliveries** across 10 Berlin depots
- **50 drivers** with varying experience levels
- **AI-powered ETA prediction** using XGBoost (78% accuracy within 15min)
- **Real-time geospatial visualization** of delivery routes
- **Driver performance analytics** and operational insights

## 📊 Key Features

### 📈 Executive Dashboard
- Real-time KPIs: delivery volume, on-time rate, satisfaction scores
- Daily volume trends and peak season analysis

### 🗺️ Interactive Map
- Depot locations and delivery points across Berlin
- Color-coded delivery outcomes (success/failure)
- Filter by date, depot, and driver experience

### 👨‍💼 Driver Analytics
- Individual driver scorecards
- Performance comparison by experience level
- Productivity and satisfaction tracking

### 🔮 ETA Prediction
- XGBoost model trained on 25+ features
- Interactive prediction widget
- Feature importance analysis

## 🏢 Business Context

Last-mile delivery accounts for **53% of total shipping costs**. This project addresses:
- ❌ Failed deliveries (wrong address, not home)
- ⏱️ Inaccurate ETAs (vague "8am-6pm" windows)
- 📉 New driver onboarding (4x faster with insights)
- 🌧️ Weather & traffic impact on routes
